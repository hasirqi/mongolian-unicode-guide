iMongol IME：传统蒙古文网页输入法

第一步先做 Web IME：用拉丁转写输入传统蒙古文 Unicode，为将来的系统输入法、聊天工具和移动端键盘打基础。

实验性说明

这是 iMongol IME 的最小可用原型。当前目标不是一次性做成标准输入法，而是先打通：转写输入 → Unicode 输出 → 复制 → 码点分析 → 竖排预览。

产品名：iMongol IME项目名：open-mongol-ime阶段：Web MVP

拉丁转写输入

当前采用最长匹配规则：先匹配 ch / sh / ng / oe / ue，再匹配单字母。后续会加入候选词、词库和更严谨的转写规则。

规则状态

  • 本地浏览器运行，不上传文本
  • 输出标准 Unicode 字符
  • 可送入码点查看器验证
  • 规则仍为实验版，需要持续校正
