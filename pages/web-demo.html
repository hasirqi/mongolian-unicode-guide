码点检测思路

function codepoints(str) {
  return Array.from(str).map(ch =>
    'U+' + ch.codePointAt(0).toString(16).toUpperCase().padStart(4, '0')
  );
}

必须显示两层

  • 用户看到的传统蒙古文文本
  • 底层 Unicode 码点序列
  • 不可见控制符位置
  • 字体与浏览器信息
